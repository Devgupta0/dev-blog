# dev-blog

> Daily technical blog posts generated by an AI agent using Llama 3 on Groq.

![Posts](https://img.shields.io/badge/Updated-Daily-brightgreen)
![AI](https://img.shields.io/badge/Written_by-Llama_3-F55036?logo=meta&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-blue)

This repo is automatically updated every day with a new markdown blog post covering real developer topics — debugging stories, system design, tools, and engineering lessons.

## Topics covered

- Production debugging and war stories
- System design and architecture
- Performance optimization
- Developer tools and workflows
- CS fundamentals explained simply

## Structure

```
posts/
├── 2024-01-15-debugging-race-condition.md
├── 2024-01-16-redis-vs-message-queues.md
└── ...
```

## How it works

An AI agent runs daily via GitHub Actions, picks a topic, generates a 700–1000 word article using Llama 3, and commits it here automatically.

Source: [github-agent](https://github.com/Devgupta0/github-agent)
